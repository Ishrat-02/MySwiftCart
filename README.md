# MySwiftCart

1) What is the difference between null and undefined?
Ans: undefined মানে হলো কোনো ভেরিয়েবলকে ডিফাইন করা হয়েছে কিন্তু তাতে কোনো মান দেওয়া হয়নি। এবং null হলো ইচ্ছাকৃতভাবে কোন ভেরিয়েবলকে খালি করা।


2) What is the use of the map() function in JavaScript? How is it different from forEach()?
Ans: map() array নিয়ে কাজ করে এবং প্রতিটি element পরিবর্তন করে নতুন একটা array রিটার্ন করে। forEach() শুধু array এর প্রতিটি element এর উপর কাজ করে, কিন্তু নতুন array রিটার্ন করে না।


3) What is the difference between == and ===?
Ans:==: মান তুলনা করে, টাইপ  দেখে না।
===: মান এবং টাইপ দুটোই ঠিকমতো মিল কিনা দেখে।


4) What is the significance of async/await in fetching API data?
Ans:async/await ব্যবহার করলে আমরা API থেকে data আনার সময় wait করতে পারি এবং কোডটা সহজে পড়া যায়।


5) Explain the concept of Scope in JavaScript (Global, Function, Block).
Ans:Scope হলো কোন ভেরিয়েবল কোন অংশে access করা যাবে সেটা।
